\# Librarian



Welcome to the Librarian repository!



Librarian is an open-access application for creating and sharing high-quality MS² reference libraries for small-molecule mass spectrometry. The three modules \*\*\*pcq\*\*\*, \*\*\*mix\*\*\* and \*\*\*lib\*\*\* bring metadata retrieval, mixture design and spectral record assembly together into a single, streamlined and customizable workflow, generating standardized records ready for deposition in repositories such as \[MassBank](https://massbank.eu).



The Librarian web application is available via https://librarian.serve.scilifelab.se/.



To run Librarian via command-line, clone the repository to your local system and run modules through librarian.py. Module commands and their use are detailed in the Supplementary Information section of the associated publication https://doi.org/10.3390/metabo16060433.

Package requirements are listed in the requirements.txt file under the app folder.
